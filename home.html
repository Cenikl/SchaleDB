Welcome, Sensei!

Schale DB is an unofficial database for the mobile game Blue Archive developed by NAT Games and published by Yostar (Japan) and Nexon (Global) for iOS and Android.

Please select your game server using the settings menu to see only information that has been released on that server.

Features
  • View Character stats, skills and profile information
  • Items, Equipment and Furniture database
  • Mission and Event drops and detailed enemy information
  • Total Assault enemy information
  • Crafting Node probabilities
Coming Soon!
  • Mission Maps
  • Shop Inventory
